## 🎯 **Quick Overview**

### What You'll Do:

1. **Create GitHub repo** (2 min)
   - Go to https://github.com/new
   - Name it "portfolio"
   - Create it

2. **Push your code** (1 min)
   ```bash
   git remote add origin https://github.com/YOUR-USERNAME/portfolio.git
   git push -u origin main
   ```

3. **Setup Hostinger** (5 min)
   - Login to Hostinger
   - Connect GitHub
   - Enable auto-deploy
   - Done!
